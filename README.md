# 🎓 Dashboard de Notas Acadêmicas

Uma calculadora de médias desenvolvida em Front-End para facilitar o cálculo da nota final do semestre, aplicando os pesos específicos das avaliações (Prova Presencial e AVA/Trabalhos).

## 🚀 Funcionalidades
- Cálculo dinâmico de médias ponderadas via JavaScript.
- Validação de campos vazios no formulário.
- Interface moderna com Dark Mode (CSS puro).
- Feedback visual de status (Aprovado/Exame) baseado no resultado matemático.

## 🛠️ Tecnologias Utilizadas
- **HTML5** - Estrutura e semântica.
- **CSS3** - Estilização e responsividade.
- **JavaScript (Vanilla)** - Lógica matemática e manipulação do DOM.
